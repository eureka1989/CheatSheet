HTML Global Attributes

Attributes give elements meaning and context.

These attributes can be used on any HTML element

Attribute name Definition Attribute value Attribute description Syntax
accesskey Specifies a shortcut key to activate/focus an element character Specifies the shortcut key to activate/focus the element
class Specifies one or more classnames for an element
Mostly used to point to a class in a style sheet or be used by a JavaScript (via the HTML DOM) to make changes to HTML elements with a specified class		 classname Specifies one or more class names for an element.
To specify multiple classes, separate the class names with a space, e.g. <span class="left important">. This allows you to combine several CSS classes for one HTML element.
Naming rules:
Must begin with a letter A-Z or a-z
Can be followed by: letters (A-Za-z), digits (0-9), hyphens ("-"), and underscores ("_")
In HTML, all values are case-insensitive
